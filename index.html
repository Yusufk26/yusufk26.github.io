Focus Forge

Focus Forge is an aim trainer game which I coded with html, js and css. This project was created for the Beta Camp buildathon in Toronto which was sponsored by the Smith School of Business. The goal of this project was to create a game which helped people with ADHD improve their focus and studying which will help them in their studies.
